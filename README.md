# BasqueNews-Dataset
A folder containing news headers, summaries and descriptions in Basque from Basque newspapers.

About the Project
-----------------
The project was created to include minoritarian languages in the latest technological advances. 
The project is open for new data submissions. It is also open to collaboration in order to diversify its topics and broaden its reach. 

Methodology
-----------------
- The data was scrapped from the net using a Python web scrapper. 
- Websites include EHU/UPV (university site), Gara (newspaper), UIK Albisteak (journal/newspaper), and Berria (newspaper)
- Some data is available in .csv, other in .txt

Uses
-----------------
The following dataset can be used for:
- Training neural models
- Basque Linguistic research
- Sociolinguistic research
